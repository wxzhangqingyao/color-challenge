# 绝对色感挑战 - 20 题终结版

一个测试用户色彩辨别能力的网页游戏，共20道题目，难度逐渐增加。

## 游戏玩法

1. 找出颜色不同的方块
2. 点击方块进行选择
3. 每道题目有不同的难度级别
4. 完成所有20道题目后查看最终成绩

## 技术栈

- HTML5
- CSS3
- JavaScript

## 部署

该项目已部署到 GitHub Pages，您可以通过以下链接访问：

[绝对色感挑战](https://<your-username>.github.io/color-challenge/)

## 本地运行

1. 克隆仓库
2. 直接在浏览器中打开 `index.html` 文件

## 游戏特性

- 响应式设计，支持移动设备
- 实时计时和错误计数
- 动态难度调整
- 完成后显示详细成绩和排名
- 流畅的动画效果
